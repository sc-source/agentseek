2. [构建你的第一个 harness 应用](02-first-harness-app.zh.md)：创建并运行可编辑项目。
3. [添加 skill 和 MCP](03-add-a-skill-and-mcp.zh.md)：添加本地行为和工具访问。

之后，用[操作指南](../how-to/index.zh.md)处理具体任务，用[参考](../reference/index.zh.md)
查准确名称。
